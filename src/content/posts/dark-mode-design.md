分享我的 **暗色模式設計原則** 🌙

好的暗色模式不只是把背景變黑，還要注意：

1. **對比度要足夠** — 確保文字在暗色背景上清晰可讀
2. **避免純黑背景** — 用深灰（如 `#16181c`）取代純黑，減少眼睛疲勞
3. **降低彩度** — 鮮豔的顏色在暗色背景上會刺眼
4. **陰影改用透明度** — 暗色主題用 `elevation` 而非 `shadow`

```css
:root {
  --bg-primary: #000000;
  --bg-secondary: #16181c;
  --text-primary: #e7e9ea;
  --text-secondary: #71767b;
}
```
